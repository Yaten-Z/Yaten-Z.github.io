原创随笔文学

你们就像天外来物一样-TATEN

2025-08-14T08:00:00.000Z2025-10-18T13:34:21.061Z

当一个我这样的人真的好累。

天空有些暗了暗的刚刚好 我难过的样子就没人看到。

在同学和老师的印象里，我是一个乐观开朗、乐于助人的好学生。在其他长辈眼中，我是乖巧懂事的孩子。在家长眼中，我是一个叛逆、没有自控力的小孩。可只有我自己知道，我是一个内向、焦虑、自卑、空虚的人。

无人在意我。每次起床的红眼，并不是得了病，而是无数个夜晚的无声的哭泣。每次去麦当当写作业，不是不喜欢家，而是渴望外界。每次与人社交，不是我很外向，而仅仅是我想要几个真正看得透我的人。

而在某一天，这一天我始终记得很清楚：2025.05.25

那是星期日下午，我早已收拾好了东西准备返校。无聊之时，我便看到了尘封依旧的知乎：99+。打开一看，映入眼帘的便是这篇改变我生活的一篇回答：很好奇现在的中学生（包括初中生，高中生）流行什么呢？ 那是我第一次认识 LINMOH.

虽说我在年头成功认识了我的两个副班长，外加几个和我性格相近的同学作为我屈指可数的朋友。但是 LINMOH 给我的感觉是不一样的：好像是一个坐在网络世界的另一边的另一个我。无论是从性格还是爱好来说，我们接近一模一样。而他也成为了我的第一个网友。

后来，我们无话不说，他还介绍了他同个城市的朋友：Susan。相比较之下，我和Susan更多是数学上的探讨。此后，我们为了巩固友谊，就建了一个群（其实是 LINMOH 不想一条信息发两次）。而这，便是我们 TATEN 的雏形。仅仅是 1 个月，我的生活就发生了巨大的变化。

随后，我们遇到了更多志同道合的朋友。我们一起在 GitHub 上建立了一个 Organization，搭建了 团队网站，做了 成员分布图。

先是 Teenmich ，在我看来他是一位非常焦虑的人，无论是学业还是生活上来看。而思维糅杂是他身上独特的地方，或许这就是我需要学习的：我常常想要书写文字以达到缓解心情、清空大脑的作用，可碍于我常常习惯整理好思路再动笔，导致我实际下来没有几篇“吐槽”。

接着，是 Errorsia ，他的出现也是我意想不到的，仅仅是一句放在曾经主页的联系方式的话（情景再现）：没想到居然有人主动想要联系我？呜呜呜太感动了qwp！

然后是更多的人加入 TATEN ，甚至最小的 member 仅仅只是小六刚毕业学生。

为何是 TATEN 这也有故事可言，它仅仅是一个意外（原因就是：键盘上 Y 和 T 太近了）。但是在此刻，我觉得 TATEN 有了更深层次的意思了：

To Always Treasure Every Node

To Achieve True Eternal Nurture

第一次意识到跨越千山万水的朋友可以只有两块显示器的距离，蜂拥而至的人们都透明了，我们便看到了彼此。无论是在脑海里，还是在内心。

而对于这些朋友，我只有一个要求：

我陪你走过的路你不能忘，因为那是我，亦或者是我们，最快乐的时光。

谨以此篇，献给我为数不多但志同道合的朋友！

谢谢你们。
