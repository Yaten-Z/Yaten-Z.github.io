CloudNS 免费注册 & Cloudflare 托管教程

发表于2025-06-05|更新于2025-06-05|信息技术
|浏览量:

一、CloudNS 免费注册

（一）访问官网

打开 CloudNS 官网：CloudNS 官网。

（二）注册账号

点击 “Sign up new account” 进行注册，输入邮箱、密码等信息。

（三）创建免费区域

  1. 登录后，进入 “DNS 托管”，点击 “创建区域”，选择 “免费区域”。
  2. 输入自定义域名（不要勾选 activate dynamic DNS），点击 “创建”。
  3. 若提示 “由于您当前 IP 网络的高滥用率，免费域名的创建受到限制…”，可更换 IP 或切换网络（你懂的）。

二、Cloudflare 托管

（一）注册 Cloudflare 账号

  1. 访问 Cloudflare 官网：Cloudflare 官网。
  2. 输入邮箱、密码，邮箱国内外均可，密码需满足一定条件。
  3. 在注册绑定的邮箱中收到邮件，点击链接激活账号。

（二）添加站点

  1. 登录 Cloudflare，点击 “主页” → “添加域”。
  2. 输入 CloudNS 申请的域名，选择 “Free” 免费计划，点击 “继续”。
  3. Cloudflare 会自动扫描已有的 DNS 记录，若有原 CloudNS 的 DNS 记录，删除后点击 “继续前往激活”。

（三）更改 DNS 服务器

  1. 在 CloudNS 中，删除所有原有 NS 记录。
  2. 添加 Cloudflare 提供的 NS 记录，主机留空，指向 Cloudflare 提供的两条 NS 服务器地址。
  3. 回到 Cloudflare，点击 “继续”，页面跳转后，拉到最下面，点击 “立即检查名称服务器”，通常 10 分钟左右，域名状态变更为 “活动”，托管成功。

三、双向解析优化方案（可选）

（一）在 Cloudflare 添加 A 记录

  1. 进入 Cloudflare → “DNS”，添加 A 记录，IP 可随意，记录类型为 A，名称如 abc，IPv4 地址可填 8.8.8.8。

（二）在 CloudNS 添加 Cloudflare 的 NS 记录

  1. 回到 CloudNS，添加两条 NS 记录，主机填写 A 记录的名称（如 abc），指向 Cloudflare 服务器地址（共两条记录）。
  2. 在 CloudNS 添加 CNAME 记录，主机为 *，指向 A 记录的域名（如 abc.cloudns.com）。

四、SSL 证书优化（可选）

  1. 在 Cloudflare → “SSL/TLS” → “边缘证书”，复制证书验证的 TXT 记录。
  2. 回到 CloudNS，添加 TXT 记录，名称为 _acme-challenge，值为 Cloudflare 提供的证书值。
  3. 等待 CloudNS 更新，Cloudflare 识别证书，通常 30 分钟内证书生效。
  4. 理论上隔一段时间要再来进行一次 SSL
