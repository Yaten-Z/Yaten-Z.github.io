从蝴蝶定理到二次曲线系（2）

发表于2025-08-16|数学学习

前文速递：从蝴蝶定理到二次曲线系（1）

在上一篇中，我们从蝴蝶定理一路推演，最终引出了「过四点的二次曲线系」的概念。为了更深入理解这种「曲线系」，我们先从更简单的直线系开始。

这一篇全是概念性的东西，熟悉的同学可以跳过。

直线系

定义 Define

在平面上，如果我们有两条不同的直线 l1,l2l_1, l_2l1​,l2​ ，它们的方程分别是：

l1:a1x+b1y+c1=0l2:a2x+b2y+c2=0l_1: a_1x + b_1y + c_1 = 0 \\ l_2: a_2x + b_2y + c_2 = 0 l1​:a1​x+b1​y+c1​=0l2​:a2​x+b2​y+c2​=0

那么我们就可以定义一组新的直线：

Lλ,μ:λ(a1x+b1y+c1)+μ(a2x+b2y+c2)=0L_{\lambda,\mu}: \lambda(a_1x+b_1y+c_1) + \mu(a_2x+b_2y+c_2) = 0 Lλ,μ​:λ(a1​x+b1​y+c1​)+μ(a2​x+b2​y+c2​)=0

这就是 一组关于 l1l_1l1​, l2l_2l2​ 的直线系。

  • 如果 l1l_1l1​ 与 l2l_2l2​ 相交于一点 PPP，那么直线系中的所有直线都经过这个点 PPP。我们称为 点束。
  • 如果 l1∥l2l_1 \parallel l_2l1​∥l2​，那么直线系中的所有直线都平行。我们称为 平行线系。

二次曲线

定义 Define

设 FFF 为定点（焦点），lll 为定直线（准线），eee 为常数且 e>0e>0e>0（离心率），则满足以下条件的动点 PPP 的轨迹称为圆锥曲线：

∣PF∣d(P,l)=e\frac{|PF|}{d(P, l)} = e d(P,l)∣PF∣​=e

其中：

  • ∣PF∣|PF|∣PF∣ 表示点 PPP 到焦点 FFF 的距离，
  • d(P,l)d(P, l)d(P,l) 表示点 PPP 到准线 lll 的距离，
  • eee 称为离心率。

根据离心率 eee 的大小，圆锥曲线分为三类：

  • 椭圆：0<e<10 < e < 10<e<1
  • 抛物线：e=1e = 1e=1
  • 双曲线：e>1e > 1e>1

极坐标方程

以焦点 FFF 为极点，极轴沿准线方向建立极坐标系，则圆锥曲线的极坐标方程为：

r=p1+ecos⁡θ或r=p1+esin⁡θr = \frac{p}{1 + e \cos \theta} \quad \text{或} \quad r = \frac{p}{1 + e \sin \theta} r=1+ecosθp​或r=1+esinθp​

其中：

  • rrr 为动点到焦点的距离，
  • θ\thetaθ 为极角，
  • eee 为离心率，
  • ppp 为极参数（半通径），与准线的位置有关。

直角坐标系下的方程

根据焦点-准线定义，可以得到二次曲线的标准方程：

  • 椭圆（焦点在 xxx 轴上，中心在原点）：

x2a2+y2b2=1,b2=a2(1−e2)\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1, \quad b^2 = a^2 (1-e^2) a2x2​+b2y2​=1,b2=a2(1−e2)

  • 抛物线（顶点在原点，开口向 xxx 轴正方向）：

y2=2pxy^2 = 2px y2=2px

  • 双曲线（中心在原点，焦点在 xxx 轴上）：

x2a2−y2b2=1,b2=a2(e2−1)\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1, \quad b^2 = a^2 (e^2 - 1) a2x2​−b2y2​=1,b2=a2(e2−1)

特点

  1. 对称性：椭圆、抛物线、双曲线都有特定的对称轴。
  2. 焦点与准线：二次曲线的几何性质可以通过焦点和准线描述。
  3. 离心率：
    • (0 < e < 1)：椭圆，轨迹闭合
    • (e = 1)：抛物线，轨迹开口
    • (e > 1)：双曲线，轨迹两条分支

二次曲线系

类似于直线系，二次曲线也可以形成一个曲线系（或称“二次曲线簇”）。

定义 Define

设有两条平面二次曲线：

C1:F1(x,y)=0,C2:F2(x,y)=0C_1: F_1(x,y) = 0, \quad C_2: F_2(x,y) = 0 C1​:F1​(x,y)=0,C2​:F2​(x,y)=0

其中 F1,F2F_1, F_2F1​,F2​ 是二次多项式：

Fi(x,y)=Aix2+Bixy+Ciy2+Dix+Eiy+Fi,i=1,2F_i(x,y) = A_i x^2 + B_i xy + C_i y^2 + D_i x + E_i y + F_i, \quad i=1,2 Fi​(x,y)=Ai​x2+Bi​xy+Ci​y2+Di​x+Ei​y+Fi​,i=1,2

那么我们可以定义一个二次曲线系：

Cλ,μ:λF1(x,y)+μF2(x,y)=0,(λ:μ)∈P1C_{\lambda, \mu}: \lambda F_1(x,y) + \mu F_2(x,y) = 0, \quad (\lambda : \mu) \in \mathbb{P}^1 Cλ,μ​:λF1​(x,y)+μF2​(x,y)=0,(λ:μ)∈P1

  • 这里 (λ:μ)(\lambda:\mu)(λ:μ) 是齐次参数，表示曲线系中的不同曲线。
  • 当 (λ:μ)=(1:0)(\lambda:\mu) = (1:0)(λ:μ)=(1:0) 时，得到 C1C_1C1​；当 (λ:μ)=(0:1)(\lambda:\mu) = (0:1)(λ:μ)=(0:1) 时，得到 C2C_2C2​。
  • 随着 (λ:μ)(\lambda:\mu)(λ:μ) 的变化，曲线系中每个参数组合都对应一条新的二次曲线。

性质

  1. 公共点：

    • 所有 Cλ,μC_{\lambda, \mu}Cλ,μ​ 都通过 C1C_1C1​ 与 C2C_2C2​ 的交点。
    • 如果 C1C_1C1​ 与 C2C_2C2​ 有 kkk 个不同交点，那么这些交点就是整个曲线系的公共点。

  2. 自由度：

    • 二次曲线一般有 5 个自由点（即通过 5 个点可以唯一确定一条二次曲线）。
    • 二次曲线系用两个固定曲线加上参数 (λ:μ)(\lambda:\mu)(λ:μ) 描述，整个曲线系是一族经过公共点的曲线。

  3. 过四点的曲线系：

    • 若 C1C_1C1​ 与 C2C_2C2​ 有 4 个不同交点 P1,P2,P3,P4P_1, P_2, P_3, P_4P1​,P2​,P3​,P4​，那么整个二次曲线系都通过这四点。
    • 这在几何证明中非常有用，例如蝴蝶定理的解析证明里，就利用了“过四点的二次曲线系”来研究曲线的交点和共线关系。
