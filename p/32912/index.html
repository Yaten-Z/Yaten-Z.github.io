发表于2025-09-22|更新于2025-09-23|博客相关
|浏览量:

前言

一直想拥有一个属于自己的小天地，能记录一些日常的思考、学习的点滴，或者单纯分享一些喜欢的东西。相比于喧嚣的社交平台，一个简洁而自由的个人博客似乎更适合。

在各种方案中，我选择了 Hexo —— 一个基于 Node.js 的静态博客框架，轻量、高效，部署方便。而在 Hexo 的众多主题里，Butterfly 则凭借着优雅的设计、强大的功能和高度可定制性，让我一眼就决定用它来搭建我的博客。

为了避免以后自己忘记步骤，也为了能给正在折腾的朋友一些参考，我决定把整个搭建过程记录成系列文章。主要会涵盖：

  • 环境准备与 Hexo 的安装

  • Butterfly 主题的下载和基础配置

  • 个性化美化和主题优化

  • More

希望这份记录不仅能帮到想要入门 Hexo 的人，也能成为我自己在折腾路上的一份“存档”。
